# component-lib-vue组件库

### 快速开始

#### 1.安装组件库

```bash
npm i component-lib-vue
```
#### 2.引用组件库
```javascript
// 按需引入
import 'component-lib-vue/dist/css/card.css
import { Card } from 'component-lib-vue'
Vue.use(Card)
```
